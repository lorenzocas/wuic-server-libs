Finestra di manutenzione: 12 mesi di aggiornamenti

La licenza consente di utilizzare tutte le versioni rilasciate fino a 1 anno dalla data di inizio licenza. Superato l'anno, la versione installata continua a funzionare indefinitamente; per passare a una versione più recente basta acquistare una licenza di estensione al costo dimezzato:

  • Developer: 300 € / anno aggiuntivo
  • Professional: 600 € / anno aggiuntivo

La licenza di estensione apre una nuova finestra di 12 mesi sugli aggiornamenti futuri. Nessun obbligo di rinnovo: si paga solo quando si vuole aggiornare.

Installazioni multiple? Fingerprint extra a 200 € / anno

Ogni licenza include un numero di machine fingerprint predefinito (1 per Developer, 5 per Professional). Se hai bisogno di deployare su più server, ogni fingerprint aggiuntivo costa 200 € / anno ed è gestito con una patch alla licenza esistente (stesso intestatario, stessa scadenza).

Esempio software house: acquisti Pro (1.200 €) + 3 fingerprint extra (600 €) = 1.800 € / anno per coprire 1 ambiente di sviluppo + 7 server cliente finali. Nessun limite tecnico al numero di fingerprint che una licenza può autorizzare.

Domande frequenti

Il prezzo include il supporto?

Licenze Developer e Professional includono accesso alla documentazione, agli aggiornamenti entro la finestra di manutenzione e al bug tracker pubblico. Il supporto dedicato (SLA email/telefono, onboarding, consulenza) è venduto separatamente.

Posso vendere progetti basati su WUIC ai miei clienti?

Sì, ed è esattamente lo scenario per cui la Pro è pensata. La licenza Professional include 5 fingerprint (tipicamente 1 ambiente di sviluppo + 4 server di clienti finali): consegni il progetto, lo installi sul server del cliente, aggiungi il fingerprint del server alla tua licenza. Se i deployment superano i 4, acquisti fingerprint aggiuntivi a 200 € / anno cadauno. Il cliente finale non ha bisogno di una propria licenza WUIC — la tua licenza autorizza il suo server.

La licenza è per sviluppatore o per macchina?

Per macchina (machineFingerprint), calcolato come hash SHA256 di nome-macchina + dominio + OS + Machine GUID. Una singola licenza può autorizzare N macchine includendo N fingerprint nel payload firmato. Il numero di fingerprint inclusi per tier è fisso (1 Dev, 5 Pro); oltre si acquistano extra a 200 € / anno cadauno.

Come aggiungo un nuovo fingerprint alla mia licenza?

Recuperi il machineFingerprint del server cliente chiamando GET /api/Meta/LicenseStatus sul suo host. Ce lo invii e ti emettiamo una licenza aggiornata con il nuovo fingerprint aggiunto (gli altri restano validi). Sostituisci license-payload e license-signature in appsettings.json, riavvii il backend, pronto.

Cosa succede se cambio server? (sostituzione, non aggiunta)

Emettiamo una licenza aggiornata con il nuovo machineFingerprint al posto del vecchio. Nessun costo aggiuntivo per sostituzione server all'interno della stessa finestra di manutenzione.

Come si passa da Developer a Professional?

In qualunque momento, pagando la differenza pro-rata in base ai mesi residui della licenza corrente. Riavvio del backend e le feature premium sono immediatamente attive.
